ezcli/root

ezcli is a minimal but flexible cli library written in C, that has self-documentation, embedded support, context passing, hooks, help generation, autocompletion, and much more.

you can safely assume any section not labelled with platform-agnostic is linux-only. and inside the actual section contents, linux-only sections are explicitly declared.

currently in v1.5.8 with ~2600 lines of code (including examples and tools).

  • if you are a first time user, you can start by reading ezcli/entrypoint
  • source code can be seen in https://github.com/alperenozdnc/ezcli
  • you can also read documentation locally if you build ezdocs
